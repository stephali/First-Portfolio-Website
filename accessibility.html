Website Accessibility & Usability

Doscument Structure
The website is styled to be consistent throughout. The menu is placed at the very top of the site for easier accessibility
For easier navigation, a simple, minimalistic style was implemented.

Screen Size
The size of the screen is compatible with any device. Text automatically resizes on the device you are using.

Visibility
The website is designed to be easy to read. Font effectively contrasts the background color to be easy on the reader's eyes

Browser Compatibility
Cascadading style sheets (css) is used for layout. Users with browsers that do not fully support css will still be able
to view the content on the page. These users will be viewing the unstyled version of the website, meaning all content will
be rendered to the browser's default color and font type.
